# Задание финального тура
Для локального запуска проекта:
```
npm install
npm run dev
```

демо по ссылке https://vercel.com/scarlettniks-projects/prod

В качестве экстрафичи выступает возможность просматривать данные на разных устрайствах, связка по аккаунтам
Геймификация - введение системы уровней без которых пользователь не может что-то покупать в магазине
Магазин - возможность купить дополнительное количество привычек и увеличить ограничение по количеству групп
Уведомления запланированы на 15:00-16:00 с интервалос в 10 минут (тестовый режим)

Отправлять уведомления можно отлько от текущего дня

После архивирования привычки она убирается с главного экрана, однако статистика по ней остается
